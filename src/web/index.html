Welcome!

This website is hosted entirely on an ESP8266 microcontroller (NodeMCU board) with just 4 megabytes of flash and 128KB of SRAM!
Yup, this entire webpage is being served by this board, including this image!

How does it work?

The ESP8266 board has 4MB of flash storage out of which we reserve 1MB for the file system. Since this entire website is little more than 15KB (excluding the image and the video), the storage is more than enough. The microcontroller connects to a local WiFi and creates a server over which the webpages and other files are served from its flash storage. There's a lot more to the process but that's the gist of it.

Still don't believe it? Learn more about how it works or you can check the source code yourself. I wrote a blog post over at my blog, The Chip Tinker Project, going into more detail and explaining how you can do this as well.
